## Decisions Made

- **Test DB strategy:** dedicated database on the same Neon project (TEST_DATABASE_URL). Cheaper to maintain than ephemeral per-run provisioning; safer than running against the dev DB. Operator must run `npm run seed:dev` against TEST_DATABASE_URL once to populate the catalog, then `npm run test:seed` to verify.
- **SEEDED_SLUGS picks:** `again-yui` (already used by tests/app.spec.ts as the canonical happy-path song), `red-swan-yoshiki-feat-hyde` (second global + lesson-complete song for parallel coverage), and `mayonaka-no-orchestra-aqua-timez` (currently geo-restricted per data/geo-audit-report.csv — supplies the regression target plan 08.1-07 needs without depending on a future replacement).
- **Single source of truth for retries policy:** `playwright.config.ts` line 19 (`retries: 0`). The grep `retries:` in playwright.config.ts now returns exactly that one config line; the docstring uses "Zero retries" wording instead of `retries: 0` so the grep stays clean for CI auditing in plan 08.1-08.
- **HTML report generated but never auto-opened:** matches the user's CLI-heavy workflow per CONTEXT.md > "Failure reporting: terminal-first summary". Operator runs `npm run test:report` only when triaging a failure.
- **`test:all` chain order:** `test:seed && test:unit && test:integration && test:qa && test:e2e`. Putting `test:seed` first means the suite aborts early on stale catalog or missing env — saves the 15-minute speed budget per CONTEXT.md.
- **Support files are not tests:** added `testIgnore: ["**/support/**"]` to playwright.config.ts and `'tests/support/**'` to vitest exclude list. Both runners would otherwise try to collect helpers as tests.
- **Excluded `tests/*.spec.ts` from vitest** (not just e2e/) — `tests/app.spec.ts` is a Playwright spec at the top level; vitest would crash on its `@playwright/test` import otherwise.

- **Found during:** Task 1 (config write)

- **Fix:** Added `'tests/*.spec.ts'` to `test.exclude` in vitest.config.ts as a defensive guard.
- **Files modified:** `vitest.config.ts`
- **Verification:** `npm test` runs only the 18 generator unit tests; no Playwright spec is collected.
- **Committed in:** `808f604`
