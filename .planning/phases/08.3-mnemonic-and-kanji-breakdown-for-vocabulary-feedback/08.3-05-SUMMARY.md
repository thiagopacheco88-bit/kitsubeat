## Operator Order-of-Operations

**IMPORTANT:** `npm run test:qa:enrichment` is designed to exit 1 until enrichment runs.

The correct order is:
1. Set `ANTHROPIC_API_KEY` in `.env.local` (https://console.anthropic.com API Keys)
2. Run `npm run seed:enrich-vocab` — enriches ~705 vocabulary_items rows (mnemonic + kanji_breakdown)
3. Run `npm run test:qa:enrichment` — should now exit 0 with `[qa-enrich] PASS`

Until step 2 is complete, the script will print `[qa-enrich] scanned N rows; N gaps` and exit 1 because all rows have `mnemonic IS NULL`. This is the intended gate behavior — not a bug.
