<objective>
Build the production-grade web-based timing editor admin tool with waveform visualization and draggable word regions.

Purpose: Per the locked decision, this is a lasting admin tool (not a throwaway dev utility) for correcting WhisperX auto-generated word timestamps. WhisperX accuracy degrades on musical audio, so every song needs at least a review pass. The editor shows the audio waveform with one draggable region per word, allowing precise timing correction.

Output: Admin pages at `/admin/timing` (song list) and `/admin/timing/[songId]` (editor), plus API routes for reading and saving timing data.
</objective>

   - Playback controls: Play/Pause button, current time display.
